# 👑 King's Store (Versi Redmi / Mobile)

**King’s Store** adalah website e-commerce yang dirancang untuk memudahkan pengguna melakukan pembelian produk secara online, baik di komputer maupun di HP.  
Tema utama website adalah **biru & putih**, menghadirkan tampilan elegan dan pengalaman belanja yang nyaman.

---

## 🌐 Link Demo (Jika Online)

> 💻 [https://kings-store.vercel.app](#) *(ganti sesuai hosting)*

---

## 📱 Fitur Mobile (Redmi & Smartphone Lain)

### 1️⃣ Halaman Utama (Landing Page)
- Navbar responsif, menu di tengah:
  - Home | Produk | Tentang Kami | Kontak
- Logo khas King’s Store
- Banner promosi yang menarik
- Tampilan menyesuaikan layar HP (responsive)

### 2️⃣ Katalog Produk
- Daftar produk dengan:
  - Gambar
  - Nama produk
  - Harga
  - Stok
- Tombol **Tambah ke Keranjang** langsung terlihat
- Contoh tabel produk:

| Produk       | Harga       | Stok |
|--------------|------------|------|
| Sepatu Sport | Rp 450.000 | 20   |
| Kaos Casual  | Rp 150.000 | 35   |
| Jam Tangan   | Rp 350.000 | 15   |
| Tas Ransel   | Rp 300.000 | 10   |

### 3️⃣ Keranjang & Checkout
- Lihat produk yang sudah ditambahkan
- Update jumlah & total harga otomatis
- Pilih metode pembayaran
- Klik **Pesan Sekarang** untuk konfirmasi pesanan

### 4️⃣ Cara Memesan di HP
1. Buka website di browser Redmi (Chrome/Mi Browser)
2. Pilih produk → klik **Tambah ke Keranjang**
3. Buka **Keranjang** → cek pesanan
4. Klik **Lanjutkan Pembayaran**
5. Isi data lengkap (Nama, Alamat, Metode Pembayaran)
6. Klik **Pesan Sekarang**, pesanan diproses

### 5️⃣ Halaman Kontak
- Formulir untuk mengirim pesan langsung
- Link media sosial (Instagram, WhatsApp, Email)
- Responsif di layar HP

## 🌟 Cara Membuka Website di HP Redmi

### 1️⃣ Buka langsung dari HP
- Salin folder website ke HP (misal `Download/KingsStore`)
- Buka file `index.html` lewat **File Manager**
- Pilih browser (Chrome, Mi Browser)
- Website akan tampil lokal di HP

### 2️⃣ Dari Server Lokal (Opsional, untuk JS/Database)
- Jalankan server di PC (XAMPP / Live Server)
- Catat IP komputer di Wi-Fi: misal `192.168.1.10`
- Di browser Redmi ketik: `http://192.168.1.10/kings-store/index.html`

### 3️⃣ Dari Hosting Online (Paling Mudah)
- Upload semua file ke hosting: Netlify, Vercel, GitHub Pages
- Buka link hosting di HP Redmi
- Website dapat diakses dari mana saja
